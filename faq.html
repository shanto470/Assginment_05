Important FAQ

What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree, allowing scripts to manipulate the document's content, structure, and style dynamically.

How do you select an element from the DOM?

To select an element from the DOM, I use use methods like: 1. document.getElementById('elementId') - Selects an element by its ID. 2. document.querySelector('selector') - Selects the first element that matches a CSS selector. 3. document.querySelectorAll('selector') - Selects all elements that match a CSS selector. 4. document.getElementsByClassName('className') - Selects all elements with a specific class. 5.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows attaching a single event listener to a parent element to manage events for its child elements, improving performance and handling dynamic content efficiently.

How do you manipulate an elements attributes and styles using the DOM?

To manipulate an element's attributes and styles in the DOM: - **Attributes:** Use `setAttribute()`, `getAttribute()`, or `removeAttribute()`. - **Styles:** Modify `element.style` or use `classList` to add/remove CSS classes.
